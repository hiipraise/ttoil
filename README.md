# Ttoil Frontend

This is the frontend project for **Ttoil**, built with [React](https://react.dev/).

## Getting Started

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Run the development server:**

   ```bash
   npm start
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
